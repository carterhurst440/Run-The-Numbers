            How It Works

            Core Loop

            1. Place one base wager.
            2. Choose COLOR, SUIT, or RANK before each draw.
            3. Pick valid values and review the multiplier.
            4. Draw one card. A match multiplies the current pot.
            5. Cash out after a hit, or continue predicting.

            Rules

            Selection Rules

            • 52-card standard deck. No Joker.
            • A fresh deck is shuffled before every draw.
            • COLOR requires exactly 1 pick.
            • SUIT allows 1 to 3 suits. RANK allows 1 to 12 ranks.
            • Multiplier is shown live before each draw and applies to the current pot on a hit.
            • Cash out pays the full base wager plus winnings minus the rung commission.
